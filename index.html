To see the loading image appear, change some content here (in the html frame) and wait a second or two.

If you prefer you can use an ajax method, you set up a page with the loading image on it, then the following code will create a response based on the destination page. However, this method probably wont work the way our page is set up because of the response.redirect (which is kinda server side ajax, right?) - can't really demo this on codepen. 

    $(document).ready( function() {
        $.ajax({
            type: 'get',
            url: 'http://destination.page.you.want.to.load',
            success: function(response) {
                // response = data which has been received and passed on to the 'success' function.
                $('body').html(response);
            }
        });
    });
